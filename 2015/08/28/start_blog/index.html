利用Github+Hexo搭建博客

发表于 2015-08-28   |   分类于 技术 ， 日常   |  

之前一直都想抽时间在github上弄一个博客，今天终于抽了一天的时间用github+hexo完成了搭建。

主要依赖的工具有：

Git
Node.js

安装Git与Node.js(Mac)

brew install git
brew install node

安装Hexo

# 安装hexo-cli相关的包
npm install hexo-cli -g 

# 初始化hexo相关的目录，生成相关目录及配置文件
# blog为生成的根目录的名字
hexo init blog 

cd blog
# 这里需要执行npm install，因为hexo init仅仅生成了一些初始化的内容，很多相关的插件与包其实是没有下载下来的，执行npm install会根据当前目录下的package.json去下载

npm install
# 启动服务，方便在本地查看效果

hexo server

发布内容到Gighub

# 安装部署文件到github的插件
npm install hexo-deployer-git --save

# 生成文件的同时部署到github
hexo generate --deploy
